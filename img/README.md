## Recomendaciones

- **Formato:** JPG o WebP para mejor compresión
- **Tamaño recomendado:** 800x600px para cards de locales
- **Tamaño hero:** 1200x800px
- **Optimización:** Comprimir imágenes para web (calidad 80-85%)
- **Nombres:** Usar minúsculas y guiones, sin espacios ni caracteres especiales
